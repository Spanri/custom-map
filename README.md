#### Installation

0. Install the npm if necessary

1. Install npm packages
   ```bash
   # npm i
   ```

#### Setup

0. Run npm command

   a. View demo, not development

   ```bash
   # npm run serve
   ```

   b. Development (auto rebuilding after `/src` changing)

   ```bash
   # npm run dev
   ```

1. Go to `http://localhost:10001/demos/`
